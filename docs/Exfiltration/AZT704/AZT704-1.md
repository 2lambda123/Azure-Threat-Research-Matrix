# AZT704.1 - Soft-Delete Recovery: Key Vault

An adversary may recover a key vault object found in a 'soft deletion' state.

!!! info "Resource" 

	Azure Key Vault
	
!!! success "Actions"

	* Microsoft.KeyVault/vaults/\*/restore
	
	* Microsoft.KeyVault/locations/deletedVaults/read
